# Anti-Patterns Flow

Detect emotional anti-patterns and dark patterns. **Never invent new coercive patterns.**

## Catalog (flag if present)

- Manufactured urgency
- Confirmshaming
- Saccharine empty-state cartoon that mocks the user
- Success confetti masking a weak outcome
- Fake social proof / trust theater
- Guilt as retention
- Anxiety as default navigation mood
- Brand voice in error messages that blocks recovery

## Output

Findings with severity, evidence, and agency-preserving alternatives.

Persist when `.productfeeling/` exists.
