# Sequence Flow

Playbooks that chain ProductFeeling commands.

## Playbooks

### New product / major surface
`init` → `jobs` → `metaphors` → `map` → `brief` → `handoff`

### Pre-launch review
`feel` or `audit` → `states` → `anti-patterns` → `review`

### Checkout / money flow
`map` → `trust` → `friction` → `tone` → `review`

### Onboarding
`jobs` → `peaks-ends` → `states` → `delight` → `brief`

### Dashboard / daily tool
`intensity` → `friction` → `tone` → `audit`

### Copy-only pass
`tone` → `anti-patterns` → `trust`

## Steps

1. Identify the situation.
2. Recommend one playbook (or adapt).
3. Run the first step now unless the user wants the list only.
4. Tell the user the remaining chain.
