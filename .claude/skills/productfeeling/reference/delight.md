# Delight Flow

Decide where intentional delight belongs — and where it must stay out.

## Rules

- Delight after earned success or during brand-led moments.
- No delight on errors, consent, destructive actions, or high-anxiety money/health/legal contexts.
- One signature delight > many generic confetti moments.
- If removing delight changes nothing about the job, it was decoration — reconsider.

## Output

Include / exclude map, signature delight proposal (if any), anti-goals.

Persist when `.productfeeling/` exists.
