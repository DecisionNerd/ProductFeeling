# Persona Flow

Run the same target through multiple emotional lenses.

## Steps

1. Select 2–4 personas or segments (include one edge / anxious / power user).
2. For each: before/during/after feeling and the most dangerous misread.
3. Find collisions — where optimizing for A harms B.
4. Recommend the primary lens and documented exceptions.

Persist when `.productfeeling/` exists.
