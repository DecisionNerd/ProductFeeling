---
title: "Chat-only mode"
description: "Use ProductFeeling without a project scaffold."
sidebar:
  order: 5
---

You do not need `.productfeeling/` or `FEELING.md` for a useful review.

1. Paste a flow, screenshot description, PRD excerpt, or copy deck.
2. Run `/productfeeling feel`, `audit`, `map`, or another technique.
3. The agent treats your chat material as context.
4. Optionally persist later with `/productfeeling init`.

A missing `FEELING.md` must never block a scoped review.
