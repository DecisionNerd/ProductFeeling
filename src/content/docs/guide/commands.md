Invoke as `/productfeeling <command> [target]`.

| Command | Purpose | Example |
|---------|---------|---------|
| `init` | Capture feeling context into docs | `/productfeeling init` |
| `feel` | General review | `/productfeeling feel onboarding` |
| `audit` | Structured audit | `/productfeeling audit checkout` |
| `critique` | Scored critique | `/productfeeling critique this PRD` |
| `review` | Pre-ship gate | `/productfeeling review billing` |
| `map` | Journey map | `/productfeeling map signup` |
| `jobs` | Emotional JTBD | `/productfeeling jobs` |
| `tone` | Microcopy feel | `/productfeeling tone errors` |
| `peaks-ends` | Peak-end design | `/productfeeling peaks-ends onboarding` |
| `friction` | Friction inventory | `/productfeeling friction settings` |
| `delight` | Delight policy | `/productfeeling delight success` |
| `trust` | Trust & anxiety | `/productfeeling trust payments` |
| `states` | UI states | `/productfeeling states empty` |
| `persona` | Segment lenses | `/productfeeling persona anxious first-timer` |
| `intensity` | Volume calibrate | `/productfeeling intensity landing` |
| `anti-patterns` | Dark patterns | `/productfeeling anti-patterns` |
| `metaphors` | Craft metaphors | `/productfeeling metaphors` |
| `brief` | Emotion brief | `/productfeeling brief dashboard` |
| `handoff` | Companion route | `/productfeeling handoff` |
| `sequence` | Handbook strategy playbooks | `/productfeeling sequence onboarding` |
| `next` | Auto-improve, no re-asking | `/productfeeling next` |
| `random` | Roll a technique | `/productfeeling random` |
| `library` | Everything you could do | `/productfeeling library` |

Pin: `/productfeeling pin audit` → `/audit`.

For depth, the skill loads **handbook chunks selectively** — it should not dump the whole handbook into context by default.
