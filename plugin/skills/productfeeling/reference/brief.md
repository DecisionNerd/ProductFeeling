# Brief Flow

Write an emotion brief Impeccable can execute without re-litigating strategy.

## Brief structure

```markdown
# Emotion brief: [surface]

## Primary feeling
...

## Supporting tones
...

## Never feel
...

## Moment map
| Moment | Feeling | Notes |
|--------|---------|-------|

## Intensity band
[1–5] + justification

## Metaphor
...

## Copy tone
Say / don't say

## Trust & anxiety
...

## Delight policy
Include / exclude

## State requirements
Empty / loading / error / success

## Out of scope for Impeccable
(feeling decisions already made; do not reopen)
```

## Steps

1. Pull FEELING.md + any prior reviews.
2. Fill every section; mark unknowns explicitly.
3. Write to `.productfeeling/briefs/<slug>-brief.md` when possible.
4. Offer `/productfeeling handoff` to route to Impeccable.
