# Critique Flow

Scored critique of a design doc, mock, PRD section, or shipped surface for emotional quality.

## Rubric (1–5 each)

1. **North-star clarity** — is the intended feeling obvious and consistent?
2. **Moment design** — peaks, endings, and recovery designed deliberately?
3. **Copy feel** — tone matches the job and audience state?
4. **Trust hygiene** — anxiety reduced without false claims?
5. **State completeness** — empty/error/loading/success covered?
6. **Ethic** — free of coercion and hollow delight?
7. **Handoff readiness** — could Impeccable execute from this alone?

## Steps

1. Read the full target.
2. Score each rubric item with evidence.
3. Name the single highest-leverage change.
4. List what already works emotionally — critique is not only negative.

## Output

Include average score, per-item table, strengths, fixes ranked, and a one-paragraph rewrite of the intended feeling if muddy.

Persist to `.productfeeling/reviews/<slug>-critique.md` when available.
