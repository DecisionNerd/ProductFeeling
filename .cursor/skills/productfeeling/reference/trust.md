# Trust Flow

Trust builders and anxiety amplifiers.

## Steps

1. List claims the UI makes (security, delivery, accuracy, privacy).
2. For each claim: what evidence appears *in-product*?
3. Map anxiety triggers (ambiguity, irreversibility, cost, social exposure).
4. Propose reducers: previews, undo, explanations, progress, human fall-backs — without false reassurance.
5. Flag trust theater (badges/copy without substance).

Persist when `.productfeeling/` exists.
