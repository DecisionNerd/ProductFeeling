# Friction Flow

Inventory friction that drains emotional energy or trust — not all UX friction, only feeling-relevant cost.

## Steps

1. List frictions with type: cognitive, emotional, temporal, social.
2. Mark which are **necessary** (security, consent) vs **accidental**.
3. For necessary friction: how to make it feel protective rather than hostile.
4. For accidental: removal or softening options.
5. Rank by impact on primary feeling.

Persist when `.productfeeling/` exists.
