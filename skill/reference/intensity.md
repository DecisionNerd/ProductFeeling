# Intensity Flow

Calibrate emotional volume: too flat vs too theatrical.

## Scale

1. Clinical / invisible
2. Quietly confident (common product default)
3. Warm and expressive
4. Brand-forward
5. Campaign / theatrical

## Steps

1. Score current intensity.
2. Score intended intensity from FEELING.md.
3. List amplifiers (motion, copy flourish, color saturation, sound) and reducers.
4. Propose a target band and 3 concrete levers.

Persist when `.productfeeling/` exists.
