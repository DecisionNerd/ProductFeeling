# Competitive landscape

Alternatives teams reach for today (not a formal market study):

| Alternative | Gap ProductFeeling aims to fill |
|---|---|
| Ad-hoc “does this feel OK?” in chat | No shared north star, no repeatable commands, no persistence contract |
| Impeccable alone | Strong craft; not specialized for emotional requirements and journey valence |
| DocSlime alone | Strong durable docs; not a feeling review skill |
| UX / psych books and checklists | Deep knowledge; not installed into the agent loop with selective handbook load |
| Dark-pattern / growth-hacking playbooks | Explicitly out of scope — ProductFeeling refuses coercion |
