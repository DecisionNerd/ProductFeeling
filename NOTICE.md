# Notice

## ProductFeeling

This project provides the **ProductFeeling** skill (`productfeeling`) for AI agents — emotion-aware product design reviews, briefs, and handoffs. Maintained under **DecisionNerd**.

## Companions

ProductFeeling is designed to sit alongside:

- [Impeccable](https://github.com/pbakaus/impeccable) by Paul Bakaus — frontend craft and execution
- [DocSlime](https://github.com/DecisionNerd/DocSlime) — documentation scaffolding and fill workflows

Not affiliated with the impeccable project.

## Architecture lineage

The skills / plugin / CLI packaging layout is modeled on [RedTeam](https://github.com/CurateLabs/RedTeam) and Impeccable.

## Disclaimer

ProductFeeling helps teams reason about intended emotional experience. It is not therapy, clinical psychology, or a license for deceptive or coercive interface patterns. Users are responsible for ethical application.
