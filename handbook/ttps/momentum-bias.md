# Momentum Bias


Give people a genuine head start without taking the decision away from them.

## What it is

Momentum bias uses visible progress, useful defaults, and editable starting points to reduce the blank-page problem. A partially prepared workspace, a suggested first task, or an honest checklist can help a user move from intention to meaningful action.

## Why it works

Starting is often harder than continuing. A clear first step and evidence of real progress lower activation energy, while progress towards a chosen goal can sustain effort. This works only when the progress is truthful and the user remains able to change course.

## When to use it

- During onboarding, when a user has chosen a goal but does not know where to begin
- When a draft, template, or `Setup Defaults` choice can save genuine effort
- When a checklist represents meaningful steps towards an outcome
- When a user returns to unfinished work and needs a respectful re-entry point

## Do

- Pre-fill only information the user expects, can inspect, and can edit
- Show progress against meaningful work, not arbitrary screens viewed
- Let users skip, remove, or change suggested steps without penalty
- Use `Gamified Progress` only when the indicator supports a real user goal

## Don't

- Mark tasks complete before the user or product has genuinely completed them
- Preselect paid, sharing, notification, or data-use choices for convenience
- Inflate a progress bar with vanity steps
- Turn streaks or unfinished work into guilt, pressure, or loss aversion

## Founder Tip

A good head start says “here is useful work already prepared for you”, not “you owe us completion”.

## Make It Yours

1. What first meaningful outcome can a new user reach quickly?
2. Which blank page could become an editable, clearly labelled draft?
3. Does every completed step represent value the user has actually received or created?
4. Which defaults require explicit choice because they affect money, privacy, or communications?
5. Can a user abandon, revise, or return later without losing their work or being shamed?
